3초 안에 끝나는 분석

FIT/GPX 파일을 놓는 즉시 파싱. 수만 개의 기록 포인트도 브라우저에서 바로 처리됩니다.

NP · IF · TSS

내 FTP를 입력하면 노멀라이즈드 파워, 강도 지수, 훈련 스트레스 점수를 즉시 계산합니다.

동기화된 인터랙티브 차트

고도·속도·파워·심박 차트 위에서 커서를 움직이면 네 차트와 지도가 같은 지점을 가리킵니다.

경로 지도 & 갤러리

GPS 트랙을 그대로 그리고, 보관함에서는 라이딩들이 경로 썸네일 갤러리로 정리됩니다. FIT를 GPX로 내보내는 것도 클릭 한 번.

SNS 공유 카드

스탯·경로·고도가 담긴 공유 이미지를 기기에서 만들어 인스타그램·카카오톡으로 바로 보냅니다.

업힐 자동 분석

오르막 구간을 자동으로 찾아 길이·경사·VAM을 계산하고 투르 방식으로 HC~4급 등급을 매깁니다.

라이더 카드 & 성장 시스템

내가 클라이머인지 스프린터인지 능력치로 판정하고, 레벨·경험치·도전과제로 게임 캐릭터 키우듯 성장을 쌓습니다.

유출 걱정 없는 분석

GPS 기록엔 집 출발지가, 심박엔 몸 상태가 그대로 담깁니다. RideLens는 그 파일을 어디로도 보내지 않으니 유출될 곳 자체가 없습니다 — 공유 링크를 만들 때만 요약본이 30일 보관 후 자동 삭제됩니다.

FAQ

자주 묻는 질문

제 파일은 정말 어디로도 안 가나요?

네. RideLens는 정적 웹 페이지이고 분석 코드 전체가 브라우저 안에서 실행됩니다. 개발자 도구의 네트워크 탭을 열어 두고 파일을 분석해 보세요 — 분석 중 파일 업로드 요청이 발생하지 않습니다.

정직한 전체 공개: 앱을 열 때 하루 최대 1회, 익명 방문 카운트 신호 1건만 자사 서버로 보냅니다. 쿠키·식별자·개인정보·라이딩 데이터는 전혀 담기지 않고("오늘 첫 방문" 같은 플래그뿐), 구글 애널리틱스 등 외부 분석 업체는 쓰지 않습니다. 자세한 내용은 앱의 [도움말 → 서비스 안내]에 있습니다.

어떤 기기의 파일을 지원하나요?

Garmin Edge·Forerunner·Fenix, Wahoo ELEMNT 등 표준 FIT 파일을 기록하는 모든 기기를 지원합니다. Strava·Komoot 등에서 내보낸 GPX 파일도 열 수 있습니다.

Strava에서 파일을 어떻게 내보내나요?

활동 페이지에서 ⋯ 메뉴 → "GPX 내보내기"를 누르면 됩니다. 원본 FIT이 필요하면 "원본 내보내기"를 선택하세요. 기기를 PC에 연결해 GARMIN/Activity 폴더에서 직접 가져와도 됩니다.

Pro를 샀는데 마음에 안 들면요?

구매 후 14일 안에는 이유를 묻지 않고 전액 환불해 드립니다. 메일 한 통이면 충분합니다.

구독은 어떻게 해지하나요? 해지하면 기록이 사라지나요?

결제 영수증 메일의 관리 링크에서 클릭 한 번으로 해지됩니다. 해지해도 무료 기능과 보관함의 라이딩 기록·레벨·배지는 전부 그대로 남습니다 — 데이터는 원래 당신 기기에만 있으니까요. 구독 자체가 싫다면 평생 소장($28.99)을 선택하세요.
