지원 파일 — FIT과 GPX의 차이

.fit — Garmin·Wahoo 등 기기가 기록하는 원본. 파워·심박·케이던스·기기 시리얼까지 담겨 있어 가장 정확합니다. 이걸 권장합니다.
.gpx — Strava·Komoot 등에서 내보낸 변환본. 경로·고도·시간 위주라 파워 데이터가 없는 경우가 많습니다. 파워가 없으면 체중·장비·경사 기반 물리 모델로 추정해 분석합니다(추정 표시가 붙음).

경험치(XP)는 어떻게 쌓이나

라이딩을 보관함에 저장하는 순간 아래 공식으로 적립됩니다:

XP = 거리(km)×10 + 획득고도(m)×0.2 + TSS×3

거리·업힐·노력 세 축이 다 반영되지만 배율이 가장 큰 건 TSS(훈련 강도)입니다. 평지 100km(약 1,500 XP)와 업힐 1,000m 포함 60km(약 1,400 XP)는 비슷하게 쌓입니다 — 짧아도 힘들게 탄 라이딩은 TSS가 높아서 XP로 보상받는 구조입니다. 결론: 더 힘든 라이딩이 더 많은 XP.

⚠️ 같은 라이딩은 한 번만 적립됩니다. 파일을 복사해 이름을 바꿔 넣어도, 다른 기기로 같은 라이딩을 기록해 넣어도 — 파일 안의 기록 시작 시각으로 판별해 중복 적립을 막습니다. 기기 여러 대의 파일을 모두 넣어도 안전합니다.

레벨·칭호·캐릭터 진화

레벨은 누적 XP로 정해집니다 (Lv = √(XP/300)+1, 상한 없음). 칭호와 아바타는 이렇게 진화합니다:

Lv.1 새싹 라이더 (맨머리) → Lv.3 동네 라이더 → Lv.5 주말 전사 (사이클캡 🧢) → Lv.8 그룹라이드 에이스 → Lv.12 지역 강자 (헬멧 ⛑️) → Lv.16 KOM 헌터 → Lv.21 엘리트 (에어로 선글라스 🕶️) → Lv.27 그랜드투어급 → Lv.34 살아있는 전설 (황금 왕관 👑 — 최고 칭호)

캐릭터(동물)는 능력치 5종 중 무엇이 강한지에 따라 정해집니다: 클라이머=산양, 스프린터=치타, 펀처=캥거루, TT 머신=송골매, 디젤=낙타, 올라운더=여우, 루키=병아리.

지표(필드) 설명 — 과학적 근거

NP (Normalized Power) — 들쭉날쭉한 파워를 생리적 부담 기준으로 환산한 평균. 30초 이동평균의 4제곱 평균(Coggan 공식, TrainingPeaks와 동일).
IF (Intensity Factor) — NP ÷ FTP. 1.0이면 1시간 전력 강도.
TSS (Training Stress Score) — 시간×강도²×100. FTP로 1시간 전력 = 100.
FTP — 1시간 유지 가능한 최대 파워. 20분 최대 파워 × 0.95로 추정.
W/kg 백분위 — Coggan 파워 프로파일 표 기반, 취미 라이더 대비 상위 %.
업힐 등급 (HC~4급) — 투르 드 프랑스 방식: 클라임 점수 = 길이(m) × 평균 경사(%).
VAM — 시간당 수직 상승고도(m/h). 프로 클라이머는 1,600+.
CTL/ATL/TSB (훈련 부하) — TSS의 42일/7일 지수가중이동평균 = 체력/피로, 그 차이 = 컨디션.
파워 추정 (파워미터 없을 때) — 공기저항(CdA)·구름저항(Crr)·경사 중력을 합산하는 표준 물리 모델. 실측 대비 ±10~15% 오차가 있어 "추정" 표시가 붙습니다.

능력치 5종: 스프린트=5초 W/kg · 펀치=1분 W/kg · 지속력=20분 기반 FTP W/kg · 클라이밍=VAM/획득고도 · 지구력=최장 거리/시간.

라이딩 공유 — 친구에게 기록 보여주기

분석 화면 상단의 "공유하기" 버튼을 누르면 나만의 라이딩 리포트를 다른 사람과 나눌 수 있습니다. 두 가지 방식을 선택할 수 있습니다:

🔗 서버 링크 — 라이딩 요약 데이터가 클라우드 임시 저장소에 30일간 보관되고, 카카오톡·메시지 등으로 보내기 좋은 짧고 깔끔한 주소가 만들어집니다. 30일이 지나면 데이터가 자동으로 삭제되어 프라이버시가 보호됩니다.
📦 오프라인 링크 — 서버에 아무것도 올리지 않고, 데이터를 브라우저에서 직접 압축해 주소 자체에 기록을 담아줍니다. 서버를 거치지 않으므로 평생 만료되지 않고 동작합니다. 다만 주소가 길어서 일부 메신저에서 잘릴 수 있습니다.

링크를 받은 상대방은 RideLens 사이트에 접속만 하면 — 회원가입·로그인 없이 — 바로 지도와 파워 분석을 볼 수 있습니다.

⚠️ 공유 시 전송되는 정보: 라이딩 이름, GPS 경로, 속도, 심박수, 케이던스, 파워, 고도 등 분석에 필요한 요약 데이터만 포함됩니다. 개인 신상 정보(이름, 연락처 등)는 포함되지 않습니다.

FAQ

내 데이터는 어디에 저장되나? — 기본은 이 브라우저의 내부 저장소입니다. 서버로는 절대 가지 않습니다. 공유 링크를 생성하는 경우에만, 사용자가 직접 버튼을 눌렀을 때 요약 데이터가 임시(30일) 저장됩니다.
브라우저 데이터를 지워도 기록을 지키려면? (강력 권장) — 보관함 → "저장 폴더 지정"으로 원하는 폴더(문서, 구글드라이브/원드라이브 동기화 폴더 등)를 한 번 지정하세요. 이후 라이딩을 넣거나 지울 때마다 그 폴더의 ridelens-backup.json이 자동으로 최신 상태로 유지됩니다. 브라우저 데이터를 지워도, 컴퓨터가 바뀌어도 이 파일만 있으면 됩니다.
복원 방법 — 새 기기/브라우저에서 보관함 → "백업 가져오기" → ridelens-backup.json 선택. 라이딩·레벨·배지·프로필·Pro 키까지 전부 복원됩니다.
수동 백업 — "백업 내보내기"를 누르면 언제든 백업 파일을 직접 저장할 수 있습니다 (자동 백업과 같은 형식).
"기록 데이터가 없어 제외"는 뭔가? — 기록 시작 직후 중지된 빈 파일입니다. 오류가 아니라 정상 제외입니다.
기기 인증 ✓는? — 보관함의 모든 라이딩이 한 기기에서 기록됐다는 표시입니다(FIT 내장 시리얼 기반). 기기가 여러 대여도 괜찮습니다 — 같은 라이딩은 기록 시작 시각으로 판별해 XP가 한 번만 적립됩니다.
